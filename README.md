# TomMaster Draw

A simple web-based drawing application built with React, TypeScript, and Vite.

## Features

- Canvas drawing with pen tool
- Color picker
- Adjustable brush size
- Clear canvas functionality
- Modern UI with Tailwind CSS

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173 to view the app.
